# Student Grade Calculator

## Overview

A Java console application that calculates a student's total marks, percentage, grade, and pass/fail result based on marks entered for five subjects.

## Features

- Accepts marks for five subjects
- Validates marks (0–100)
- Calculates total marks
- Calculates percentage
- Displays grade
- Displays pass/fail result

## Concepts Used

- Variables
- Data Types
- Scanner Class
- Arithmetic Operators
- if-else-if Ladder
- Methods
- Input Validation
