# Number Guessing Game

## Overview
A console-based Java game where the computer randomly generates a number between 1 and 100. The player has five chances to guess the correct number with hints provided after each incorrect guess.

## Features
- Random number generation (1–100)
- Five guessing attempts
- Hints after each guess (Too High / Too Low)
- Displays remaining chances
- Displays the correct number if all chances are used

## Concepts Used
- Variables
- Data Types
- Scanner Class
- Random Class
- if-else Statements
- for Loop
- Methods
