# Java Basics Projects

## Overview

This folder contains beginner-friendly Java projects created while learning the fundamentals of Java programming. Each project focuses on strengthening core concepts through hands-on implementation rather than just theory.

## Projects

### 1. Calculator
A menu-driven calculator that performs basic arithmetic operations.

**Concepts Covered:**
- Scanner Class
- Variables
- Data Types
- Arithmetic Operators
- switch-case
- do-while Loop

---

### 2. Number Guessing Game
A console-based game where the user guesses a randomly generated number within a limited number of attempts.

**Concepts Covered:**
- Scanner Class
- Random Class
- Methods
- if-else Statements
- Loops

---

### 3. Temperature Converter
A menu-driven application that converts temperatures between Celsius, Fahrenheit, and Kelvin.

**Concepts Covered:**
- Scanner Class
- Methods
- Arithmetic Expressions
- switch-case
- do-while Loop

---

### 4. Student Grade Calculator
A program that calculates total marks, percentage, grade, and pass/fail status based on marks entered for five subjects.

**Concepts Covered:**
- Scanner Class
- Input Validation
- Variables
- if-else-if Ladder
- Methods
- Arithmetic Operations

---
