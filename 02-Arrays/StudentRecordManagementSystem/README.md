# Student Record Management System

## Overview

A menu-driven Java console application that manages student records using arrays. The application allows users to add students, view student details, search students by name, and display class statistics.

## Features

- Add multiple student records
- View all student records
- Search student by name (case-insensitive)
- Display class statistics
  - Total students
  - Highest marks
  - Lowest marks
  - Average marks
- Input validation for:
  - Number of students
  - Student marks

## Concepts Used

- Arrays
- Parallel Arrays
- Methods
- Scanner Class
- Loops
- Conditional Statements
- Searching
- Input Validation
