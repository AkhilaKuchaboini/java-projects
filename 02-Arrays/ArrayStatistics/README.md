# Array Statistics

## Overview

A Java console application that accepts an array of integers and performs common statistical operations such as calculating the sum, average, largest element, and smallest element.

## Features

- Accepts user-defined array size
- Reads array elements from the user
- Displays all array elements
- Calculates the sum of all elements
- Calculates the average
- Finds the largest element
- Finds the smallest element

## Concepts Used

- Arrays
- Scanner Class
- for Loop
- Enhanced for Loop
- Variables
- Arithmetic Operations
- Conditional Statements
